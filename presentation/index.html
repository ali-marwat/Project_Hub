🎓 Project Hub

A unified platform for university students to discover, share, and collaborate on academic projects.

Presented by: Ali Raza

The Problem We Solve

😩 Problems Faced

  • ❌ No centralized platform to view past student projects
  • ❌ Students repeat similar ideas without knowing prior work
  • ❌ Hard to discover quality code & project inspiration
  • ❌ No admin oversight to ensure project quality
  • ❌ No community interaction — no feedback or collaboration

✅ Our Solution

  • 🌐 Centralized hub where all projects live in one place
  • 🔍 Search & filter to discover existing work before starting
  • ⭐ GitHub integration to showcase real, quality code
  • 🛡️ Admin panel with approval/rejection workflow
  • 💬 Comments, upvotes, and community collaboration

Key Features

🔐

GitHub OAuth Login

One-click sign-in with GitHub. Profile photo auto-synced across the platform.

📦

Smart Repo Import

Fetch your GitHub repos, auto-detect languages, stars, forks — then submit in 4 steps.

🛡️

Admin Moderation

Admins approve or reject with reasons. Rejected projects show feedback to students.

📊

Language Statistics

GitHub-style colored language bar auto-generated for each project from the API.

💬

Comments & Upvotes

Real-time commenting, upvoting, and community interaction on every project.

🔍

Search & Filter

Filter by category, sort by popularity or date, with animated expandable pill tabs.
